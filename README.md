# World Clock - Multiple Time Zones

A beautiful and functional digital clock application that displays the current time in different time zones around the world.

## Features

✨ **Key Features:**
- 🌍 Display current time in multiple time zones simultaneously
- 🕐 Digital and analog clock displays
- ➕ Add/remove time zones easily
- 🔍 Search functionality for quick filtering
- 💾 Save your selected time zones (localStorage)
- 📱 Fully responsive design
- 🎨 Modern gradient UI with smooth animations
- 🔄 Real-time updates every second

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start adding time zones you want to monitor

## Default Time Zones

The application comes with three default time zones:
- **New York** (America/New_York)
- **London** (Europe/London)
- **Tokyo** (Asia/Tokyo)

## How to Use

### Adding a Time Zone
1. Click the **"+ Add Time Zone"** button
2. Search for your desired city or region
3. Click on the time zone to add it
4. The clock will appear in the main grid

### Removing a Time Zone
- Click the **×** button on any clock to remove it

### Searching Clocks
- Use the search box at the top to filter displayed clocks by city name

## Features Explained

### Digital Display
Shows the exact time in HH:MM:SS format for precise time tracking.

### Analog Clock
Traditional analog clock representation for visual time reading.

### Timezone Offset
Displays the UTC offset (e.g., UTC+05:30) for each timezone.

### Date & Day
Shows the current date and day of the week in the selected timezone.

### Local Storage
Your selected time zones are automatically saved in your browser, so they persist between sessions.
